## Requirements

- iOS 11+
- Swift 5+



## Installation

### CocoaPods

```
pod 'GEAudioPlayer', '0.1.0'
```

### Carthage

```
github "Goeun1001/GEAudioPlayer"
```

### SPM

```
.package(url: "https://github.com/Goeun1001/GEAudioPlayer.git", from: "0.1.0")
```
